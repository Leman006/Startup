# Dashboard Client

React приложение на Vite для дешборда.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

## Сборка

```bash
npm run build
```

## Просмотр production сборки

```bash
npm run preview
```
